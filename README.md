# Classic Digital Printing Website

Professional photo lab website for Classic Digital Printing in Lahore, Pakistan.

## 🚀 Features

- **Intro Animation**: Camera lens-inspired loading animation
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Glass Morphism UI**: Modern glassmorphism effects
- **Smooth Animations**: CSS keyframe animations and transitions
- **WhatsApp Integration**: Direct order form to WhatsApp
- **Image Gallery**: Portfolio showcase with modal viewer
- **Customer Reviews**: Auto-scrolling testimonials
- **Google Maps**: Embedded location map

## 📁 Project Structure

```
classicdigitalstudio/
├── index.html          # Main HTML file
├── css/
```

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom animations, flexbox, grid
- **Tailwind CSS**: Utility-first CSS framework (CDN)
- **JavaScript (Vanilla)**: No dependencies
- **Google Fonts**: Inter font family

## 🎨 Design Features

### Color Palette
- Primary: Amber (#f59e0b)
- Background: Black (#000)
- Glass: rgba(30, 30, 35, 0.6)
- Text: White (#fff) / Gray variants

### Animations
- `focusLens`: Logo reveal animation
- `spinRing`: Rotating ring effect
- `slowZoom`: Background zoom
- `scroll-left/right`: Review carousel
- `fade-in-up`: Content reveal

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## 🔧 Setup & Development
